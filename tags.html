Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be dangerous. In the process of making it. we'll learn hou to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1 –h6 Headings Include a heading (levels 1 – 6)
p Paragraph Include a paragraph of text
table Table Include a table
tr Table row Include a row of data
th Table header Make a table header
td Table data Include a table data cell
header Header Represents a container for introductory content
div Divider Defines a section in a document
ol Ordered list Defines an ordered list in a document
ul Unordered list Defines an unordered list in a document

Inline Elements

Tag Name Purpose Example Result
em Emphasized Make emphasized text <em>techincal_sophistication</em> techincal_sophistication
strong Strong Make strong text <strong>at least a billion people</strong> at least a billion people
a Anchor Make an hyperlink <a href="http://learnenough.com">Learn Enough</a> Learn Enough
img Image Display an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
code Code Define some text as computer code in a document <code>rails server</code> rails server
span Span Mark up a part of a text, or a part of a document. <p>My mother has <span style="color:blue">blue</span> eyes.</p>

My mother has blue eyes.
li List element Defines a list item. <li>Salty sailors</li>
